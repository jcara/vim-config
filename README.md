# vim-config

Here's my configuration for vim.  

## How to install

### 1. Clone repository

```
    git clone --recursive https://github.com/jcara/vim-config.git ~/.vim
```

### 2. Optional - backup your .vimrc
```
    cp ~/.vimrc ~/.vimrc.orig
```

### 3. Copy .vimrc
```
    cp vimrc ~/.vimrc
```

### 4. Create directories
    mkdir -p ~/.vim/colors


### 5. Copy theme.
```
    cp vim-monokai/colors/monokai.vim ~/.vim/colors/monokai.vim
```
